## Installation
1. Create a python virtual enviroment:  
	`python -m venv .venv`

2. Activate the python venv if necessary.  
    Windows: `.\.venv\Scripts\activate`   
    Linux: `source ./.venv/bin/activate`  

3. Install dependencies:  
	`pip install -r requirements.txt`

4. Run the project:   
	`python main.py`
